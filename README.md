# 🗺️ Mapa Conceitual

> *"O conhecimento não existe na mente de forma isolada — ele se organiza em redes de conceitos interligados."*
> — Joseph D. Novak

**Repositório dedicado à criação e organização de mapas conceituais**, uma ferramenta visual de aprendizagem que representa graficamente conceitos e as relações entre eles.

</div>

---

## 📖 Sobre o Projeto

**Mapas conceituais** são diagramas que organizam e representam o conhecimento de forma hierárquica, conectando ideias por meio de **palavras de ligação** que explicam o relacionamento entre os conceitos.

Essa técnica, desenvolvida por **Joseph Novak** na década de 1970 na Universidade de Cornell, é amplamente reconhecida na educação por:

- 🧠 **Facilitar a compreensão** de conteúdos complexos
- 🔗 **Evidenciar relações** entre conceitos distintos
- 📚 **Otimizar o estudo** e a retenção de informações
- 🏗️ **Estruturar o conhecimento** de forma hierárquica e visual

Este projeto reúne mapas conceituais elaborados para auxiliar no estudo e na organização de diferentes temas, promovendo uma visão clara e estruturada do conhecimento.

## 🧩 Estrutura de um Mapa Conceitual

```
        [ Conceito Central ]
               │
    ┌──────────┼──────────┐
    │          │          │
[Conceito] [Conceito] [Conceito]
    │
[palavra de ligação]
    │
[Conceito Filho]
```

| Elemento | Descrição |
|---|---|
| **Conceito** | Ideia ou objeto representado dentro de um nó/caixa |
| **Palavra de ligação** | Verbo ou locução que conecta dois conceitos |
| **Proposição** | Unidade semântica formada por dois conceitos + palavra de ligação |
| **Hierarquia** | Organização dos conceitos do mais geral ao mais específico |
| **Links cruzados** | Conexões entre conceitos de ramos diferentes |
