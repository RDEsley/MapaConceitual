# Mapa Conceitual

Repositório dedicado à criação e organização de **mapas conceituais**, uma ferramenta visual de aprendizagem que representa graficamente conceitos e as relações entre eles.

## Sobre

Mapas conceituais são diagramas que organizam e representam o conhecimento de forma hierárquica, conectando ideias por meio de palavras de ligação que explicam o relacionamento entre os conceitos. Essa técnica, desenvolvida por Joseph Novak na década de 1970, facilita a compreensão, o estudo e a memorização de conteúdos complexos.

Este projeto reúne mapas conceituais elaborados para auxiliar no estudo e na organização de diferentes temas, promovendo uma visão clara e estruturada do conhecimento.
